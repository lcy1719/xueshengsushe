## 功能介绍
**基于SpringBoot+Vue实现的学生宿舍信息管理系统采用前后端分离的架构方式实现，系统设计了管理员、商家、维修人员、学生4种角色，系统有前台、管理系统两套页面，管理端实现的功能有用户管理、商家管理、维修人员管理、宿舍信息管理、宿舍分配管理、桶装水管理、失物招领管理、系统管理等功能。**
